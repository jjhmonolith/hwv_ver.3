# HW Validator ver.3 - 페이지별 상세 기획

## 페이지 구성 요약

### 공통
- `/` - 홈 (교사/학생 선택)

### 교사 플로우
- `/teacher/login` - 로그인/회원가입
- `/teacher/dashboard` - 세션 관리 대시보드
- `/teacher/sessions/[id]` - 세션 상세 및 참가자 관리
- `/teacher/sessions/[id]/qr` - QR 코드 표시

### 학생 플로우
- `/join` - 접근 코드 입력
- `/join/[code]` - 세션 참가 (이름/학번 입력)
- `/interview/upload` - PDF 파일 업로드
- `/interview/start` - 인터뷰 모드 선택
- `/interview` - 인터뷰 진행 (채팅/음성)
- `/interview/transition` - 주제 전환 대기
- `/interview/complete` - 결과 페이지
